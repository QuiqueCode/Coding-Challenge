# IMPORTANTE

## Nota sobre el `.env`

El archivo `.env` fue dejado de forma **intencional** en el repositorio para facilitar la ejecución del proyecto durante la evaluación.

Normalmente este archivo no debería subirse a un repositorio (por buenas prácticas de seguridad), sin embargo, en este caso consideré mejor dejarlo para facilitar la evaluación

```env
DATABASE_URL="file:./dev.db"
```

## Instalación y ejecución

1. Clona el repositorio e instala las dependencias:

```bash
npm install
```

2. Levanta la base de datos local con datos de prueba:

```bash
npm run db:setup
```

3. Inicia el servidor de desarrollo:

```bash
npm run dev

```
